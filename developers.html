Developer Resources

Switchboard V2​

switchboardv2-api​

switchboardv2-api: Anchor generated typescript client to interact with the Switchboard program.

Documentation: /api/ts/

Source Code: github.com/switchboard-xyz/switchboardv2-api

Install: Run the following command

  • npm
  • Yarn
npm install @switchboard-xyz/switchboardv2-api
yarn add @switchboard-xyz/switchboardv2-api

switchboardpy​

switchboardpy: Python client to interact with Switchboard V2.

Documentation: /api/py

Source Code: github.com/switchboard-xyz/switchboardpy-api

Install: Run the following command

pip install switchboardpy

switchboard-v2​

switchboard-v2: Rust crate to interact with Switchboard V2 from an on-chain program.

Documentation: crates.io, docs.rs

Source Code: github.com/switchboard-xyz/switchboardv2-rs

Install: Add the following line to your Cargo.toml

[dependencies]
switchboard-v2 = "0.1.8"

switchboardv2-cli​

switchboardv2-cli: Command line interface to interact with Switchboard V2.

Documentation: /api/cli

Source Code: github.com/switchboard-xyz/cli

Install: Run the following command

  • npm
  • Yarn
npm install -g @switchboard-xyz/cli
yarn add -g @switchboard-xyz/cli

switchboardv2-example​

switchboardv2-example: Example demonstrating the full path from creating an oracle queue and aggregator, then running an oracle and updating it's result.

Documentation: github.com/switchboard-xyz/switchboard-v2-example

Source Code: github.com/switchboard-xyz/switchboard-v2-example

Install: Run the following command

git clone https://github.com/switchboard-xyz/switchboard-v2-example
cd switchboard-v2-example
npm install
